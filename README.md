  ### The work

  I like the seam where craft meets curiosity — interfaces that feel honest, APIs that don't ask too much of you, and
  the occasional excuse to learn something new because the problem said so.

  These days that pulls me in a few directions at once: React and Next.js on the front, Node and Express on the back,
  Flutter for things people carry in their pockets, Solidity when a contract is genuinely the right answer, and a steady
   diet of ML fundamentals and AI development for everything in between.

  ### The stack

  **Languages** — TypeScript · JavaScript · Python · Dart · Solidity
  **Frontend** — React · Next.js · HTML · CSS · Tailwind · Bootstrap
  **Backend** — Node.js · Express
  **Mobile** — Flutter
  **AI / ML** — ML fundamentals · AI development
  **Design** — Figma · Graphic design
  **Tools** — Postman · Git

  ### How I work

  - Treat the design and the code as the same problem from two angles.
  - Write things down — that's why I keep a [digital garden](https://github.com/BENIMANA-II/portfolio).
  - Build, watch what people actually do, change one thing at a time.
  - Stay curious about adjacent disciplines — the best ideas usually come from the next field over.

  ### What's next
